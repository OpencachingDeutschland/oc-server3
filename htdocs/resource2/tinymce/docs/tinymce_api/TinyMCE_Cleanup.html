addRuleStr

void addRuleStr(<string> s)
    Adds a cleanup rule string, for TinyMCE_Cleanup.prototype.example = a;[href|name|title=title|class=class1?class2?class3]. These rules are then used when serializing the DOM tree as a HTML string, it gives the possibility to control the valid elements and attributes and force attribute values or default them.

      s - Rule string to parse and add to the cleanup rules array.
